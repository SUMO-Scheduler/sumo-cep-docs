Mapper is an object-to-object service, which allows you to solve issues with mapping of properties in one object of one type to another object of another type.

INFO: This is a class group page. You can provide a class group page for classes grouped togehter using the @group tag with the @group-content tag. See the tags documentation for more info.

Mapper implementation rules:

- mapper has two types of methods: void (mutable) and non-void (non-mutable)

- mapper name and mapper methods map* part should be the same.

class SomethingMapper {

    /* good non-void method */

    static Anything mapSomethingToAnything(Something source) {}

    /* good void method */

    static void mapSomethingToAnything(Something source, Anything target) {}

    /* good method */

    static Anything mapSomethingSObjectToAnything(Something__c source) {}

    /* bad method because it belongs to AnythingMapper */

    static Something mapAnythingToSomething(Anything source, Something target) {}

}
